Why I took this course

I took this course to learn how to build clean, accessible, and professional websites using HTML and CSS. I wanted to learn not just how websites look, but how they were functioned and structured behind the scenes. Throughout this course, I became better at working with semantic HTML, layouts, and responsive designs, I was always interested in UX/UI design, so learning web development helped me understand how to create user-friendly sites. Overall, this course gave me practical skills and a stronger foundation in web development.

How to use this site

Each topic page includes (1) an embedded YouTube video, (2) a blog post or article link, and (3) short descriptions explaining why each resource is helpful.
